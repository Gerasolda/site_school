# HabitFlow — Трекер Привычек

**Первый пет-проект**  
*Простой, но полноценный трекер привычек на чистом HTML, CSS и Vanilla JavaScript*

![HabitFlow Preview](https://via.placeholder.com/800x400/10b981/ffffff?text=HabitFlow+Preview)

## О проекте

**HabitFlow** — это минималистичный и красивый трекер привычек, разработанный как **первый пет-проект** для отработки навыков Frontend-разработки.

Проект демонстрирует уверенное владение:
- HTML5 и семантической разметкой
- Современным CSS (Tailwind + кастомные стили, CSS-переменные, тёмная тема)
- Vanilla JavaScript (ES6+)
- Модульным подходом к архитектуре
- Работой с `localStorage`
- Адаптивной вёрсткой (mobile-first)

---

## Основной функционал

### Управление привычками
- Добавление новой привычки
- Редактирование и удаление привычек
- Поддержка двух типов периодичности:
  - Ежедневно
  - Выбранные дни недели (Пн, Ср, Пт и т.д.)

### Отслеживание прогресса
- Отметка выполнения привычки за текущий день (toggle)
- История выполнения по датам

### Календарь активности
- Интерактивный календарь на месяц
- Цветовая индикация:
  - Зелёный — выполнено
  - Красный — пропущено
  - Серый — нет данных

### Статистика и геймификация
- Текущий **streak** (серия дней подряд)
- Процент выполнения за последние 30 дней
- Общая система очков за выполнение привычек
- Глобальная статистика

### Дополнительно
- Поддержка светлой и тёмной темы
- Адаптивный дизайн (отлично выглядит на мобильных устройствах)
- Плавные анимации и микровзаимодействия
- Toast-уведомления
